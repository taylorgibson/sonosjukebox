## Introduction

The hardware used in this project consists of:
* Rasberry Pi 3B with power supply and flash card
* [Waveshare PN532 NFC HAT](https://www.waveshare.com/wiki/PN532_NFC_HAT#Read_NTAG2XX_card)
* NTAG215 NFC tags
